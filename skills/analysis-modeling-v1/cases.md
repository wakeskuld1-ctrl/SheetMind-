## Scenario 13: user asks for short-term warning forecast

### User
"先看未来几周有没有预警风险。"

1. Confirm `time_column` + `value_column`.
2. Route to `short_term_forecast_alert`.

### Skill should not do

## Scenario 14: user asks who drove period-over-period change

### User
"本月相对上月，谁贡献了变化？"

## Scenario 15: user asks what-if comparison before action

### User
"如果提价 3% 和砍投放 10%，哪种更好？"

### Skill should do
1. Confirm target metric and driver columns.
2. Build 2-3 scenarios with `driver_changes`.
3. Route to `scenario_simulation` and explain projected range + uncertainty.

### Skill should not do
- Should not present deterministic business commitment from one simulation run.
